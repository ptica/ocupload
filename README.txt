Install:
=============================
1. Extract in sites/all/modules
2. Enable module BUEditor one click upload
3. Add new button to your BUEditor with code: "js: E.showFileSelectionDialog();" (screenshot: http://xandeadx.ru/sites/default/files/images/part-4/20110311145005.png)
4. (Optional) Edit templates in admin/config/content/bocu
